How to Use

1. Prepare the data
1.1 Download the dataset (ZIP file) from GitHub
1.2 Create a folder and name it ML
1.3 Extract all files from the ZIP and put them into the ML folder

2. Using the tool
2.1 First, load and train the model
2.1.1 Click the Browse button on the tool’s UI
2.1.2 Select the ML folder you prepared (just click it once—no need to open it)
2.1.3 Click Train Model

3. Making predictions
3.1 Go to the prediction page on the UI
3.2 Upload an audio file or a spectrogram image you want to analyze
3.3 Click the process button to run the prediction
